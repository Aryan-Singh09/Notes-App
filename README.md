📝 Notes App

A simple and elegant Notes App built using Python and Tkinter. This application allows users to create, open, edit, and save text notes through a clean dark-themed graphical interface.

🚀 Features
📄 Create new notes
📂 Open existing .txt files
💾 Save notes as .txt files
🌙 Dark mode interface
📜 Scrollable text editor
⌨️ Keyboard shortcuts
⚠️ Error handling for file operations
🚪 Exit confirmation dialog
🔤 UTF-8 encoding support
🛠️ Technologies Used
Python 3
Tkinter (GUI Library)

🎮 Keyboard Shortcuts
Shortcut	Action
Ctrl + N	Create New Note
Ctrl + O	Open Note
Ctrl + S	Save Note
Ctrl + Q	Exit Application

💡 How It Works
Launch the application.
Start typing notes in the editor.
Use the File menu or keyboard shortcuts to:
Create a new note
Open an existing note
Save your work
Exit safely using the Exit option or Ctrl + Q.
